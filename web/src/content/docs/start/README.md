# Start

This section is the front door. Three pages, in order: what Net is and where it fits, a five-minute quickstart that gets a bus running and consuming events, and the install guide for every supported language.

Read [What is Net?](./what-is-net) first if you're new to the system. Skip straight to the [Quickstart](./quickstart) if you've already heard the pitch and want to see code. The [Install](./install) page covers the package names and per-language setup if you need to wire up a non-Rust binding.
